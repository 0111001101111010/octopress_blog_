Installing VirtualBox on Ubuntu 13.04

  1. When installing virtual box on linux boxes generic headers need to be installed.

    1. go to vagrantup and grab the latest version for your os
    2. install latest version
    3. install linux-generic headers suggestion by virtualbox
    4. Virtual Box
    5. Restart is necessary or same error will be thrown

More at

http://askubuntu.com/questions/171684/trouble-running-virtualbox-on-ubuntu

Using PPAs

  1. Clear benefit of using PPAs instead of ones provided by linux canonical

    1. most up to date version
    2. fixed anything if you are doing early_adopter/bleeding edge projects
    3. usually add once, and done

head to launchpad.net
read technical instructions
add that PPA for your version and flavor of linux

sudo apt-get update
sudo apt-get upgrade

Either now you will have an updated version of your tool you are trying to run or you can now sudo apt-get install it
